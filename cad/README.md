# Sheng CAD Starter

Status: L2 V5 build-packet candidate support only.

This folder exists to anchor V5 CAD/provenance work without implying that
fabrication geometry is ready.

Current authority:

- `p0-p1-control-parameters.csv` is a design-table starter for review and
  measurement planning.
- `mcp-session-log.md` records the provenance and authority class of source
  artifacts.

Current non-authority:

- No DXF is cut-ready.
- No CAD body, windchest, reed window, socket map, or gasket land is
  fabrication authority.
- No Wolfram output has been executed or exported as runtime evidence.

Promotion beyond L2 requires measured P0 reed coupon data, measured P1
single-pipe control data, and reviewed DXF/CAD/design-table authority for the
chosen branch.
